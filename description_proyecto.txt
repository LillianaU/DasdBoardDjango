# Descripción del Proyecto Django

Este documento describe los pasos para crear un proyecto Django llamado `MenuSpa` con una aplicación `spa`.

## Paso 1: Instalar Django
Primero, asegúrate de tener Python instalado en tu sistema. Luego, instala Django usando pip:
```sh
pip install django


Paso 2: Crear el Proyecto Django
Crea un nuevo proyecto Django llamado MenuSpa:
django-admin startproject MenuSpa

Paso 3: Navegar al Directorio del Proyecto
Cambia al directorio del proyecto recién creado:
cd MenuSpa
Paso 4: Crear una Aplicación Django
Dentro del proyecto, crea una nueva aplicación llamada spa:

python  startapp spa

Paso 5: Configurar la Aplicación en settings.py
Abre el archivo MenuSpa/settings.py y añade la aplicación spa a la lista INSTALLED_APPS:

INSTALLED_APPS = [
    # ... otras aplicaciones ...
    'spa',
]

Paso 6: Configurar URLs
Configura las URLs del proyecto en MenuSpa/urls.py para incluir las URLs de la aplicación spa:
from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path('admin/', admin.site.urls),
    path('spa/', include('spa.urls')),
]
Crea un archivo urls.py en la aplicación spa y define algunas rutas básicas:
# filepath: spa/urls.py
from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
    # Agrega más rutas según sea necesario
]

Paso 7: Crear una Vista Básica
Define una vista básica en spa/views.py:
# filepath: spa/views.py
from django.shortcuts import render

def index(request):
    return render(request, 'spa/menu.html')
Paso 8: Crear Plantillas HTML
Crea una carpeta templates/spa dentro de la aplicación spa y añade archivos HTML como base.html, menu.html, palindromos.html, resta.html, y suma.html. Por ejemplo, menu.html podría verse así:
<!-- filepath: spa/templates/spa/menu.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Menu</title>
</head>
<body>
    <h1>Bienvenido al Menú</h1>
    <!-- Contenido del menú -->
</body>
</html>
Paso 9: Migrar la Base de Datos
Realiza las migraciones iniciales para configurar la base de datos:
python  migrate
Paso 10: Ejecutar el Servidor de Desarrollo
Finalmente, ejecuta el servidor de desarrollo para verificar que todo funciona correctamente:
python  runserver
Estructura del Proyecto
Después de seguir estos pasos, tu estructura de proyecto debería verse similar a la que has mostrado:
MenuSpa/

manage.py: Utilidad de línea de comandos para tareas administrativas.
MenuSpa/settings.py: Configuraciones del proyecto.
MenuSpa/urls.py: Enrutamiento de URLs del proyecto.
spa/urls.py: Enrutamiento de URLs de la aplicación.
spa/views.py: Vistas de la aplicación.
spa/templates/spa/: Plantillas HTML de la aplicación.

iniciar servidor de la aplicación
python manage.py runserver
